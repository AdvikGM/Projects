What is Coding?

Coding is like a Recipe. You give the computer a list of steps, and it follows them exactly.

The 3 Main Jobs of Code:

1. Remembering Things (Variables)

Coding lets the computer "hold" information in its memory, like a player's name or a high score.

Example: Score = 100

2. Making Choices (If/Then)

Coding tells the computer what to do when something happens.

Example: IF it is raining, THEN show an umbrella icon.

3. Doing it Again (Loops)

Coding is great at doing boring tasks over and over without getting tired.

Example: Send a "Happy Birthday" text to ALL 50 friends at once.
